# Personal Finance Tracker

A full-stack application for tracking personal income and expenses with role-based access, analytics, and JWT authentication.

## Tech Stack

- **Frontend:** React 18, Vite, React Router, Axios, Recharts
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **Caching:** Redis
- **Authentication:** JWT
- **API Docs:** Swagger (OpenAPI 3)
